# JS: DOM y Eventos: To-Do List

Manipular el DOM dinámicamente y manejar eventos de usuario para construir una lista de tareas interactiva.

### Instrucciones

1. Implementa el diseño en: 
Agregar nuevas tareas desde un campo de entrada
Marcar tareas como completadas
Eliminar tareas
Mostrar un contador de tareas pendientes

### Requisitos:
**1. Debes implementar tu solución utilizando únicamente JavaScript puro (sin librerías)**
2. Asegúrate de usar los siguientes métodos de manejo del DOM:
addEventListener
createElement
appendChild
classList

### Consideraciones
Evita recargar la página ni redireccionar al usuario
Mantén una estructura clara entre lógica y manipulación del DOM
Usa nombres de variables descriptivos
**No se debe utilizar ningún framework**
